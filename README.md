# GiveSmarter

Monorepo for a GiveSmart-class fundraising platform.

## Quick start
1. `npm install`
2. `npm run dev`

## Docs
- `docs/plan.md` build plan and workstreams.

## Structure
- `apps/web` Next.js app (public site, donor portal, admin).
- `packages/*` shared domain, DB, UI, workflows, tests.
